## Getting Started

1. Clone repo & install dependencies:
```bash
npm install
```

2. Add `.env` file with:
```
VITE_SANITY_PROJECT_ID=your_project_id
```

3. Start dev server:
```bash
npm run dev
```

4. Run Sanity studio:
```bash
cd sanity
npm install
npm run dev
```
